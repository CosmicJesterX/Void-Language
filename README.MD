# ☁️ VOIDLANG ☁️

> _"Hva om språket ditt bare kjørte når du ikke så på?"_

Velkommen til **Voidlang**, et *språk* (eller kanskje en tilstand) som **ikke finnes**... før du kjører det.  
Det er ikke laget for produksjon. Det er laget for... *resonans.*

---

## 🤷 Hva er dette?

Voidlang er en idé.  
En manifestasjon av et behov for å *gjøre noe rart*,  
mens du drikker te og venter på at CI skal feile.

Du trenger ikke kunne koding.  
Bare følg stemningen.

---

## 🧪 Eksempel på Voidlang-ish

```voidlang
ritual open_portal {
  require "🫖"
  whisper "Åpner portalen til dev/null"
  drift {
    if (te_is_ready) {
      merge ::void
    } else {
      echo "VENT PÅ TEEN"
    }
  }
}
